# 用户门户需求摘要

## 模块概述

用户工作流门户是低代码工作流平台面向最终用户的操作界面，提供流程发起、任务处理、工作台管理等核心功能。

## 核心功能模块

### 1. 个人工作台
- Dashboard概览界面（任务概览、流程概览、个人绩效、可视化图表）
- 个性化设置（主题、通知偏好、工作偏好、语言设置）
- 可拖拽组件网格（任务概览、流程统计、快捷操作、通知中心组件）

### 2. 流程管理
- 流程发起（分类导航、搜索筛选、发起对话框、表单填写）
- 流程跟踪（我的申请列表、状态筛选、流程撤回、流程催办）
- 流程统计分析（发起统计、参与统计、效率分析）
- 流程详情页面（四区域布局：表单数据+流程图+流转详情+操作按钮）

### 3. 任务管理
- 待办任务（多维度分配：直接用户、虚拟组、部门角色、委托任务）
- 任务处理（权限验证、动作执行、附件管理）
- 任务认领机制（虚拟组认领、部门角色认领、认领冲突处理）
- 任务历史（处理记录、绩效评估）

### 4. 委托管理
- 委托设置（委托类型、委托条件、委托对象、委托生效管理）
- 代理处理（代理任务标识、权限限制、操作记录）
- 委托监控（状态监控、委托审计）

### 5. 权限申请
- 权限申请流程（功能权限、数据权限、审批流程）
- 权限查看（权限树形结构、使用统计）
- 权限生命周期（到期提醒、续期申请、权限回收）

### 6. 消息通知
- 通知中心（消息分类、通知展示）
- 通知渠道（站内通知、邮件通知）
- 通知设置（个人偏好、频率控制）

## 技术栈

### 前端
- Vue 3.4.0 + TypeScript 5.0.0
- Vue Router 4.2.0 + Pinia 2.1.0
- Element Plus 2.4.0
- ECharts 5.4.0（数据可视化）
- Form-Create 3.1.0（动态表单）
- BPMN.js 17.0.0（流程图显示）

### 后端
- Java 17 + Spring Boot 3.x
- 复用 workflow-engine-core 的流程引擎服务
- 复用 admin-center 的用户、组织、角色服务

## 设计规范

### 汇丰主题配色
- 主导航：汇丰红色（#DB0011）
- 卡片组件：白色背景，浅灰色边框（#E5E5E5）
- 数据展示：成功绿色（#00A651），警告橙色（#FF6600），错误深红色（#CC0000）

### 布局规范
- 顶部导航栏：高度60px
- 左侧菜单栏：宽度240px，可折叠
- 主内容区：白色背景，圆角卡片布局

## 非功能需求
- 首屏加载 < 2秒
- 操作响应 < 500ms
- 支持100 TPS
- 支持三语言（英文/中文简体/中文繁体）
