# 开发者工作站需求

## 1. 概述

开发者工作站是低代码工作流平台的核心开发工具，为业务开发人员和技术人员提供可视化的流程设计、数据建模、表单设计和业务逻辑配置功能。

### 1.1 目标用户
- **业务分析师**：设计业务流程和表单
- **技术开发人员**：配置数据模型和集成逻辑  
- **流程设计师**：创建和优化工作流程
- **系统管理员**：管理功能单元和版本

### 1.2 核心价值
- **可视化开发**：通过拖拽和配置减少编码工作
- **快速原型**：快速构建和验证业务流程
- **版本管理**：完整的功能单元生命周期管理
- **标准化**：统一的开发规范和最佳实践

## 2. 界面设计规范

### 2.1 整体布局
开发者工作站采用经典的IDE布局设计，包含以下区域：

**顶部标题栏**（高度60像素）：
- 左侧显示平台Logo和名称"工作流开发平台"
- 中间显示当前功能单元名称和状态
- 右侧显示用户信息、通知图标、个人设置入口和退出登录按钮

**左侧导航栏**（宽度240像素，可折叠）：
- 主导航菜单：工作台、功能单元、图标库、使用说明、设置
- 快捷操作按钮：新建、导入、克隆、导出功能单元
- 最近使用项目列表：显示最近编辑的功能单元

**主内容区域**（占据剩余空间）：
- 面包屑导航：显示当前页面路径
- 页面工具栏：包含页面相关的操作按钮
- 内容区域：显示具体的功能页面
- 状态栏：显示当前操作状态和提示信息

### 2.2 视觉设计规范

**汇丰银行主题配色系统**：
- 主色调：汇丰红色（#DB0011）用于主要操作按钮、重要提示和品牌标识
- 辅助色：深灰色（#333333）用于标题文字，中灰色（#666666）用于正文
- 成功色：绿色（#00A651）表示成功状态和完成操作
- 警告色：橙色（#FF6600）表示警告状态和需要注意的信息
- 错误色：深红色（#CC0000）表示错误状态和危险操作
- 背景色：纯白色（#FFFFFF）主背景，浅灰色（#F8F9FA）辅助背景

**现代化字体系统**：
- 中文字体：优先使用"PingFang SC"（苹果系统）、"Microsoft YaHei"（Windows系统）、"Noto Sans CJK SC"（Linux系统）
- 英文字体：优先使用"Helvetica Neue"、"Arial"、"Segoe UI"
- 等宽字体：代码和数据显示使用"Consolas"、"Monaco"、"Courier New"
- 字体大小：标题24px/20px/18px，正文16px，说明文字14px，小字12px
- 行高：正文1.5倍行高，标题1.2倍行高，确保良好的阅读体验

**OA布局设计原则**：
- 采用经典的办公自动化系统布局：顶部导航栏 + 左侧菜单 + 主内容区
- 界面层次清晰：使用卡片、分割线、阴影等元素区分不同功能区域
- 信息密度适中：避免信息过于密集，保持适当的留白空间
- 操作流程清晰：重要操作使用明显的视觉引导和状态反馈

**图标规范**：
- 使用统一的图标库，保持视觉一致性
- 图标风格：线性图标为主，填充图标为辅
- 图标大小：16px、20px、24px、32px四种规格
- 图标颜色：与汇丰主题配色保持一致

### 2.3 登录系统

**登录界面设计**：
- 居中显示的登录卡片，宽度400像素
- 顶部显示平台Logo和标题
- 用户名和密码输入框，支持记住密码功能
- 多因素认证支持：短信验证码、邮箱验证码
- 忘记密码链接和找回密码功能

**认证流程**：
1. 用户输入用户名和密码进行基础认证
2. 系统检查用户是否启用多因素认证
3. 如启用MFA，发送验证码到用户手机或邮箱
4. 用户输入验证码完成二次验证
5. 验证成功后生成JWT令牌并跳转到主界面

**安全措施**：
- 密码输入框支持显示/隐藏切换
- 连续登录失败5次后锁定账户30分钟
- 支持图形验证码防止暴力破解
- 登录日志记录，包含IP地址和时间戳

## 3. 主导航和功能菜单

- 功能描述：显示个人工作概览和快速访问入口

**功能单元管理**：

- 功能描述：管理系统中使用的所有图标资源

- 预览功能：图标网格显示，支持搜索和筛选

**使用说明**：
- 功能描述：提供平台使用帮助和操作指南

### 3.2 搜索功能

**全局搜索**：
- 搜索范围：功能单元名称、描述、标签、文档内容
- 搜索方式：实时搜索，输入关键词即时显示结果

- 高级搜索：支持筛选条件和排序方式

**搜索结果展示**：
- 结果列表：显示匹配的项目，包含标题、描述、类型标签
- 高亮显示：搜索关键词在结果中高亮显示
- 快速跳转：点击搜索结果直接跳转到对应页面
- 搜索历史：记录最近的搜索关键词

**常用操作快捷键**：
- 新建功能单元：Ctrl+N

- 导入功能单元：Ctrl+I
- 导出功能单元：Ctrl+E
- 复制选中元素：Ctrl+C
- 粘贴元素：Ctrl+V
- 撤销操作：Ctrl+Z
- 重做操作：Ctrl+Y

**快捷操作面板**：
- 位置：左侧导航栏下方
- 内容：最常用的操作按钮，如新建、导入、导出
- 自定义：用户可以自定义显示的快捷操作
- 徽章提示：显示相关数量，如草稿数量、待发布数量

## 5. 图标资源管理系统

### 5.1 图标库架构

**图标分类体系**：
- 系统图标：系统内置图标，包含流程、表单、动作、状态、导航相关图标

**图标选择器组件**：
- 触发方式：点击图标选择按钮弹出选择对话框

**图标预览和导出**：

## 6. 功能单元管理

### 6.1 功能单元概念

**功能单元定义**：
功能单元是平台的基本业务单位，包含完整的业务流程实现。每个功能单元包括：
- 一个工作流程：通过流程设计器创建的BPMN流程
- 数据表结构：一张主表和若干子表、动作表
- 用户界面：主表单和动作表单
- 业务动作：默认动作和自定义动作

**组件关系**：
- 功能单元是顶层容器，包含所有组件
- 表提供数据存储结构，生成DDL用于数据库创建
- 表单提供用户界面，绑定到表和流程节点
- 动作定义用户操作，绑定到流程步骤
- 工作流连接表单和动作，实现完整的业务流程

### 6.2 功能单元界面

**列表显示**：
- 显示方式：网格卡片布局，每个功能单元占一个卡片
- 卡片内容：图标（64x64像素）、名称、版本、状态标签、描述文字
- 状态显示：草稿（灰色）、已发布（绿色）、已废弃（红色）
- 排序方式：按创建时间、修改时间、名称、状态排序

**操作功能**：
- 创建新功能单元：点击标题栏"创建新功能单元"按钮
- 编辑功能单元：点击卡片进入编辑界面
- 发布功能单元：仅草稿状态显示，点击后确认发布
- 导出功能单元：仅已发布状态显示，下载功能单元包
- 删除功能单元：点击后显示确认对话框
- 克隆功能单元：复制现有功能单元创建新版本

### 6.3 功能单元编辑

**编辑界面布局**：
- 标签页导航：流程设计器、表设计器、表单设计器、动作设计器
- 顶部信息栏：显示功能单元名称、状态、版本信息
- 操作按钮：保存、发布、设置、删除按钮
- 设置对话框：编辑图标、版本、名称、状态、描述等基本信息

**状态管理**：
- 编辑状态：可以修改所有内容，但不能导出
- 发布状态：内容只读，可以导出和部署
- 版本控制：每次发布创建新版本，保留历史版本
- 变更跟踪：记录每次修改的内容和操作人员

## 7. 流程设计器

### 7.1 BPMN流程设计器

**设计器界面**：
- 左侧工具面板：包含BPMN元素工具箱（事件、活动、网关、连接线）
- 中央画布区域：流程图绘制区域，支持拖拽和连接操作
- 右侧属性面板：显示选中元素的属性配置
- 顶部工具栏：文件操作、编辑操作、视图操作、验证操作

**BPMN元素支持**：
- 事件类：开始事件、结束事件、中间事件、边界事件
- 活动类：用户任务、服务任务、脚本任务、子流程
- 网关类：排他网关、并行网关、包容网关、事件网关
- 连接类：顺序流、消息流、关联线

**画布操作**：
- 元素创建：从工具箱拖拽元素到画布
- 元素连接：拖拽元素边缘的连接点创建连接线
- 元素选择：单击选择、Ctrl多选、拖拽框选
- 元素编辑：双击进入快速编辑模式
- 画布导航：缩放、平移、适应窗口、实际大小

### 7.2 节点配置

**用户任务配置**：
- 基本信息：任务名称、任务描述、任务文档链接
- 分配方式：指定用户、指定角色、指定部门、表达式分配
- 任务属性：优先级、到期时间、是否可委托、是否可转办
- 表单绑定：选择与任务关联的表单
- 动作绑定：选择任务可执行的动作

**网关条件配置**：
- 条件表达式：为每个出口设置条件表达式
- 默认路径：设置网关的默认执行路径
- 表达式编辑器：可视化的表达式编辑工具
- 条件测试：支持条件表达式的测试验证

### 7.3 流程验证和测试

**实时验证**：
- 结构完整性：检查流程是否有开始和结束事件
- 连接有效性：检查所有元素是否正确连接
- 网关条件：检查网关是否配置了必要的条件
- 任务分配：检查用户任务是否配置了分配规则
- 表单绑定：检查任务是否绑定了有效的表单

**流程模拟**：
- 测试场景：创建测试用例，设置初始变量和模拟操作
- 逐步执行：按步骤执行流程逻辑，高亮当前节点
- 执行日志：详细记录每步的执行情况和变量变化
- 结果分析：执行时长统计、路径覆盖分析、异常记录

## 8. 表设计器

### 8.1 表设计器界面

**主界面布局**：
- 左侧表列表：显示当前功能单元的所有表（主表、子表、动作表）
- 中央编辑区域：使用类似Excel的表格界面编辑表结构
- 右侧属性面板：配置选中字段的详细属性
- 底部状态栏：显示当前表信息和DDL生成状态

**表格编辑功能**：
- 字段管理：添加、删除、移动字段，支持拖拽排序
- 数据类型：VARCHAR、INTEGER、DECIMAL、DATE、TIMESTAMP、BOOLEAN、TEXT、BLOB
- 字段属性：长度、精度、小数位、允许空值、默认值、主键、唯一、索引、注释
- 约束管理：主键约束、外键约束、唯一约束、检查约束

### 8.2 表关系设计

**父子表关系**：
- 关系类型：一对多关系，主外键关联
- 关系配置：选择父表、外键字段、关联字段、级联操作
- 循环检测：实时检测并阻止循环依赖的形成
- 关系图：可视化显示表之间的关系

**约束和索引**：
- 主键约束：单字段或复合主键，自动创建唯一索引
- 外键约束：指定引用表和字段，配置级联操作
- 唯一约束：单字段或复合唯一约束
- 普通索引：提高查询性能的索引

### 8.3 DDL生成和测试

**DDL自动生成**：
- CREATE TABLE：根据表结构生成建表语句
- ALTER TABLE：比较版本差异生成增量更新语句
- 数据库方言：支持MySQL、PostgreSQL、Oracle、SQL Server
- 命名规范：自动处理表名和字段名的命名规范

**DDL测试验证**：
- 语法验证：检查生成的DDL语句语法正确性
- 连接测试：连接到开发环境的测试数据库
- 事务执行：在事务中执行DDL进行测试
- 自动回滚：测试完成后自动回滚，不影响数据库
