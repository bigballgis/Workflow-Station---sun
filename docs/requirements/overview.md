# 低代码工作流开发和运行平台需求文档

## 项目概述

本平台是一个面向HSBC内部的企业级低代码工作流开发和运行平台，旨在提高业务流程的数字化效率，降低开发成本，加快业务响应速度。平台采用模块化设计，支持可视化开发、灵活配置和企业级部署。

### 业务背景
- **目标用户**：HSBC内部业务部门、IT开发团队、系统管理员
- **业务价值**：快速构建和部署业务流程，减少重复开发工作，提高业务敏捷性
- **应用场景**：审批流程、业务申请、数据收集、任务管理等各类企业内部流程

### 平台架构

本平台由三个核心模块组成，形成完整的开发-管理-使用闭环：

#### 1. 开发人员工作站（Developer Workstation）
**用途**：功能单元的设计和开发
- 可视化流程设计器（基于BPMN 2.0）
- 数据表结构设计器
- 表单界面设计器
- 业务动作配置器
- 功能单元版本管理和发布

#### 2. 管理员中心（Admin Center）
**用途**：系统配置和权限管理
- 用户和组织架构管理
- 角色权限配置
- 功能单元部署和管理
- 系统监控和审计
- 数据字典和关联表管理

#### 3. 用户工作流门户（User Portal）
**用途**：最终用户的流程操作界面
- 流程发起和处理
- 任务管理和委托
- 个人工作台和仪表板
- 流程跟踪和历史查询

### 核心概念

#### 功能单元（Function Unit）
功能单元是平台的基本部署单位，包含：
- **工作流定义**：基于BPMN 2.0的流程模型
- **数据模型**：主表、子表、动作表的完整数据结构
- **用户界面**：与流程节点绑定的表单配置
- **业务逻辑**：默认动作和自定义动作的配置

#### 设计理念
- **低代码开发**：通过可视化设计器减少编码工作
- **配置驱动**：通过配置而非硬编码实现业务逻辑
- **模块化架构**：功能单元可独立开发、测试、部署
- **企业级特性**：支持权限控制、审计跟踪、高可用部署

### 技术栈概览

#### 前端技术
- **框架**：Vue 3 + TypeScript
- **流程设计**：bpmn.js
- **表单引擎**：form-create
- **表格组件**：Handsontable
- **UI组件库**：Element Plus / Ant Design Vue

#### 后端技术
- **开发语言**：Java 17+
- **框架**：Spring Boot 3.x
- **工作流引擎**：Flowable 7.x
- **数据库**：PostgreSQL 14+
- **文件存储**：MinIO
- **缓存**：Redis
- **消息队列**：Apache Kafka 7.x

#### 部署和运维
- **容器化**：Docker + Docker Compose
- **编排**：Kubernetes（可选）
- **监控**：Prometheus + Grafana
- **日志**：ELK Stack

### 项目范围

#### 包含功能
- 完整的低代码开发工具链
- 企业级权限管理系统
- 工作流执行引擎
- 用户友好的操作界面
- 基础的系统监控和审计

#### 不包含功能
- 复杂的报表和BI功能
- 第三方系统集成（初期）
- 移动端原生应用
- 高级的AI/ML功能

### 成功标准

#### 功能性指标
- 支持完整的BPMN 2.0流程元素
- 支持20+种表单字段类型
- 功能单元从开发到部署时间 < 1天
- 用户界面响应时间 < 2秒

#### 非功能性指标
- 支持1000+并发用户
- 系统可用性 > 99.5%
- 数据安全符合企业标准
- 通过安全审计和合规检查
